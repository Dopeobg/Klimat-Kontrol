# Климат Контроль — лендинг

## Как посмотреть

### Вариант 1 (самый простой)
- Откройте файл `index.html` в браузере.

### Вариант 2 (рекомендуется, чтобы всё работало как на хостинге)
- В PowerShell из папки проекта:

```powershell
python -m http.server 5173
```

- Откройте `http://localhost:5173/`.

## Где менять контент

- **Фон HERO**: `assets/hero-bg.png`
- **Телефон / Telegram**: `index.html` и `js/main.js`
- **Галерея работ** сейчас сделана в виде премиальных плейсхолдеров. Если у вас есть реальные фото — положите их в `assets/` и замените блоки в секции `#works`.
